# AI Moments 演示手册

> 本 App 为交互原型，AI 分析、视频编码、相册写入均为模拟。

## 开场 30 秒（建议话术）

「这是 AI 相册 Demo。用户平时拍照，AI 在后台从相册里自动发现旅行、日常等高光，生成约 8 秒的 Vlog，并提醒用户查看和分享。今天我们重点看**从相册到成片**的完整体验。」

## 路线 A：最快核心链路（约 2 分钟）

适合时间紧的评审。

1. 打开 App → 相册页顶部有 **「演示」** 标记
2. 点 **「① 创作链路」**（已预填 5 个素材、8 秒时长）
3. 底部 **「下一步」** → 创作设置（可强调「视频长度偏好 8 秒」）→ **保存并开始创作**
4. 等待 AI 分析（演示模式约 5 秒）→ 自动进入剪辑页
5. 点播放预览 → 底部切换「文字 / 音频」→ 返回剪辑
6. **保存到相册** → **分享作品** → 结束

**强调一句话：** 全程用户只选素材和偏好，编排、卡点、字幕由 AI 完成。

## 路线 B：成果先行（约 1 分钟）

适合先展示「做完之后什么样」。

1. 相册页 → **「② 看成果」**
2. 展示已生成 Vlog、高光动态照片、处理摘要
3. 点编辑 / 下载 → 进入剪辑或结果页
4. 再回到相册，补充说明「这是 AI 次日提醒后用户看到的样子」

## 路线 C：完整故事（约 5 分钟）

适合产品细节评审。

1. 相册 → 右侧 **AI 小精灵** → 了解引导
2. 引导中可随时点右上角 **「跳过体验」** 进入助手首页
3. **AI 高光助手** → 大按钮「开始 AI 创作」
4. 接路线 A 第 3 步起
5. 最后进 **作品中心** 看任务列表与分享

## 演示前检查

- [ ] 使用含 `DemoMode = true` 的演示包
- [ ] 首页能看到「演示快捷入口」卡片
- [ ] 次日提醒卡片已显示（可点进成果页）
- [ ] 视频时长各处显示为 **8 秒**（设置、剪辑、导出一致）

## 常见问题应答

| 提问 | 建议回答 |
|------|----------|
| 视频为什么不是真播放？ | 当前是 UI 原型，播放为模拟；接后端后可换真实预览 |
| 保存进相册了吗？ | 演示包不做真实写入，导出页为成功态模拟 |
| 和剪映有什么区别？ | 我们强调**从系统相册自动发现高光**，不是手动导入剪辑 |
| 隐私怎么说？ | 引导里有本地优先 / 云端辅助选项，用户可控 |

## 关闭演示模式

正式发布前，将 `AppState.kt` 中 `DemoMode` 改为 `false`。
